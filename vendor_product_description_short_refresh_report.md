## Methodology
For each product, three description variants were generated:
- **Variant A:** Style first, benefit second
- **Variant B:** Benefit first, style second
- **Variant C:** Ultra-minimal, premium

Each variant was tested against:
1. ✅ **Length test:** ≤ 240 characters
2. ✅ **Clarity test:** No unnecessary jargon
3. ✅ **Truth test:** Only claims from series_description or product specifics
4. ✅ **Value test:** At least 1 style cue + 1 practical benefit (except Variant C)
5. ✅ **Spam test:** No ALL CAPS, fake scarcity, or over-hype
